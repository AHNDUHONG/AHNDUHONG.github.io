훈련 세트와 테스트 세트 분리

((42,), (14,), (42,), (14,))

  • reshape(-1)은 무슨 의미인가?

    • x.reshape(-1)은 x.reshape(1, -1)과 같이 1차원 배열을 반환합니다.
    • x.reshape(-1, 1) => shape(12, 1)
    • x.reshape(-1, 2) => shape(6, 2)
    • x.reshape(-1, 3) => shape(4, 3)

    ++++ 출처: https://rfriend.tistory.com/345 [R, Python 분석과 프로그래밍의 친구 (by R Friend)]
